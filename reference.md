# PM PRD Copilot 参考索引

本文件只作为入口索引。详细规则已拆分到 `references/`，避免主参考文件继续膨胀。

| 文件 | 何时读取 |
| :--- | :--- |
| [references/section-menu.md](references/section-menu.md) | 需要默认骨架、现有逻辑写法、全局规格协同或安全规则时 |
| [references/prd-workflow-strategies.md](references/prd-workflow-strategies.md) | 需要判断新写 PRD、修改已有 PRD、独立增量 PRD 或仅评审已有 PRD 的策略时 |
| [references/clarification-and-review.md](references/clarification-and-review.md) | 需要澄清追问、结构性问题处理、四视角自检或问题分级时 |
| [references/revision-records.md](references/revision-records.md) | 需要新增或编辑 PRD 修订记录时 |

使用原则：

- `SKILL.md` 负责主 workflow、硬规则和确认门。
- `references/` 负责细则、清单和可选流程。
- 不默认一次性读取所有参考文件；只读取当前任务需要的文件。
